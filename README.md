# removeAIer-med-popular-zh--AI-skill

> 专为 AI Cli工具（Claude Code、OpenClaw、Codex 等）设计的中文健康科普降AI率智能体技能（Skill）

本项目提供了一个高度专业化的 Skill 配置文件，旨在帮助医生、医学科普作者和运营人员将机器生成的健康科普文章、患者教育材料，转化为**有温度的真实医生讲解体**，彻底消除“AI机械味”，同时确保医学事实和临床建议的绝对安全。

## ✨ 核心特性

- 🧑‍⚕️ **医生语感重塑**：将机械的AI输出转为“医生讲解体”，叙事节奏慢下来，让读者能跟得上、看得懂。
- 🛡️ **反健康焦虑机制**：硬性约束改写后**不得对读者产生比原文更强的焦虑感**，精准清除“沉默杀手”“不容忽视”等危言耸听的流量词。
- 🧩 **打破三段式套路**：拒绝 AI 万能的“病因→症状→预防”流水账结构，重构自然流畅的科普逻辑。
- 📱 **多平台风格自适应**：执行前会主动询问发布平台（公众号/小红书/患教手册/医院官网等），自动匹配对应的受众语感。
- 🎯 **模糊建议具体化**：强制将“适量运动”“合理饮食”等无法执行的废话，转化为“每周150分钟”“每天少吃一两克盐”等可操作指导。

## 📂 适用文档范围

- 微信公众号、小红书、科普平台的健康科普文章
- 患者教育手册、出院宣教材料、随访指导单
- 疾病介绍、用药指导、术后护理科普
- 医院官网科普栏目、健康管理 App 内容
- 健康类短视频脚本、图文科普内容

## ⚙️ 工作原理

本 Skill 采用严格的顺序执行机制，单次调用仅执行一轮，防止模型“越级”导致文本崩坏：

### 第 1 轮：词汇叙事层人性化改写
打破“AI 说明书体”。通过词汇降维（将“干预”转为“处理”、“预后”转为“后续恢复”）、叙事慢速化（增加通俗解释和过渡句）、类比具体化（把抽象损伤转为“水管结水垢”式的具象描述），建立真实人类的讲述感。

### 第 2 轮：健康科普 AI 模式检测与清除
在第 1 轮基础上，进行深度的“排雷”与“重构”：
1. **清除焦虑煽动**：删除“正在悄悄侵蚀你的健康”等恐吓式营销话术。
2. **破除三段式公式**：打乱刻板的“病因-症状-预防”并列结构，用具体问题或场景切入。
3. **具名化归因**：将万能的“研究表明”强制替换为具体期刊数据或直接陈述。
4. **删除空洞励志**：清除“只要配合治疗一定能战胜疾病”等毫无临床依据的免责式鼓励。
5. **具体化建议**：将“保持良好作息”等正确的废话，改为有具体量化指标的行动指南。

## 🚀 如何使用

### 1. 安装配置
将本仓库中的 `SKILL.md` 文件放置到你的 AI 助手 Skill 读取目录下：

- **Claude Code**: 通常放置于项目根目录的 `.claude/skills/` 或直接放在工作区根目录。
- **OpenClaw**: 放置于 `.claw/skills/` 或根据具体配置的 `skill_path` 映射。
- **Codex / 其他兼容工具**: 放置于智能体能够读取上下文的 `./skills/` 或 `./prompts/` 文件夹中。

### 2. 触发 Skill
在对话中输入以下任意关键词，智能体将自动加载此 Skill，**并首先询问您文章发布的平台**，随后默认执行第 1 轮：

- `健康科普降AI`
- `患教材料去AI味`
- `疾病科普文章人性化改写`
- `公众号医学文章降AIGC`
- `科普推文去机器感`
- `患者教育内容改写`

### 3. 两轮交互流程
1. **输入文本 + 触发词** ➡️ 智能体询问平台 ➡️ 回答平台（如“微信公众号”）➡️ 获得第 1 轮改写结果。
2. **在同一对话中回复**：“继续执行第2轮” ➡️ 获得最终清除 AI 痕迹与套话的版本。

## 🛑 安全与约束声明

- **零幻觉原则**：绝对禁止新增任何医学事实、诊断标准、用药剂量或临床建议。
- **方向锁定**：不得改变原文对疾病预后、用药风险或生活方式建议的判断方向。
- **防焦虑底线**：改写后的文章绝不允许比原文更具恐吓性或煽动性。
- **语体守卫**：走向“医生朋友讲大白话”，但绝不走向“网络白话腔”或“主持人腔”（禁止使用“咱们来看看”“今天来聊聊”）。
- **纯净输出**：不输出任何说明性文字、润色建议或 Markdown 粗体（除非原文有）。
